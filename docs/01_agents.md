# Agent Catalog

This catalog summarizes the reusable agents included in the framework and explains how each one can be adapted for the Greenscale reference implementation.

| Agent | Purpose | Typical Inputs | Typical Outputs | Greenscale Example |
|---|---|---|---|---|
| Customer Support Agent | Handles support requests and routes unresolved issues | Customer message, user profile, support policy | Reply draft, escalation note | Respond to partner/vendor questions |
| Document Q&A Agent | Answers questions using uploaded documents | User question, documents, retrieval results | Cited answer, source references | Search research or operations documents |
| Executive Assistant Agent | Helps with scheduling, summaries, reminders, and prioritization | Calendar, email, tasks, notes | Daily brief, meeting prep, action items | Prepare leadership updates |
| KPI Monitoring Agent | Tracks metrics and flags changes | KPI data, thresholds, reporting cadence | Alert, dashboard summary, weekly report | Monitor growth, research, or operations metrics |
| Market Research Agent | Collects and summarizes market information | Topic, competitors, sources, constraints | Market brief, trend summary | Track AgTech and sustainability trends |
| Competitive Intelligence Agent | Monitors competitors and strategic risks | Competitor list, news, websites, filings | Competitor brief, risk summary | Track relevant companies and products |
| Lead Qualification Agent | Scores and routes potential leads | Contact details, company info, messages | Lead score, recommendation, CRM update | Prioritize partnership opportunities |
| Meeting Notes Agent | Summarizes meetings and extracts action items | Transcript, agenda, attendees | Summary, decisions, action items | Summarize research or business meetings |
| Report Generation Agent | Converts research and data into structured reports | Notes, KPIs, research summaries | Executive report, weekly report | Create Greenscale progress reports |
| Multi-Agent Coordinator | Routes tasks across multiple agents | User request, task type, available agents | Agent selection, workflow plan | Coordinate research → synthesis → report generation |
