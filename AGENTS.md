## Project Purpose

This repository supports building, documenting, and deploying no-code AI agents. The goal is to make agent workflows understandable, reusable, and easy to maintain for non-technical users.
Agent Instructions
When working in this repository:

```bash
Preserve beginner-friendly explanations.
Avoid unnecessary code unless it is clearly required.
Prefer step-by-step workflows, checklists, templates, and examples.
Keep terminology consistent: “agent,” “workflow,” “tool,” “trigger,” “action,” and “output.”
Do not remove documentation that helps non-technical users understand the system.
```
