# Agent Examples

Each example shows how an agent handles a complete task from input to structured output.

## Standard Files

- `input.json` — example request or event
- `routing.json` — why the coordinator selected this agent
- `tools_used.json` — tools or data sources used
- `output_schema.json` — expected structured response format
- `sample_output.json` — example successful result
- `failure_modes.md` — known failure cases and fallback behavior

## Examples

| Agent | Purpose |
|---|---|
| Customer Support | Handles customer issues and support tickets |
| Document Q&A | Answers questions from uploaded documents |
| Employee Support | Answers internal HR or policy questions |
| Executive Assistant | Summarizes calendar, email, and prep tasks |
| KPI Monitoring | Tracks metrics and flags operational risk |
| Market Research | Produces structured external research |
| Meeting Notes | Extracts summaries, decisions, and action items |
| Coordinator | Routes complex requests across multiple agents |
