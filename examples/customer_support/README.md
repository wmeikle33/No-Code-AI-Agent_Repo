# Case Study: Refund Request Escalation

## Business Problem

Customers frequently contact support requesting refunds.
Many requests can be handled automatically, but some require
manager approval due to company policy.

## User Request

"I purchased the premium subscription yesterday and would like a refund."

## Selected Agent

Customer Support Agent

## Why Coordinator Selected This Agent

The request was classified as:

- Customer inquiry
- Refund related
- Requires policy lookup

The routing confidence score was 0.96.

## Retrieved Knowledge

Sources used:

- refund_policy.md
- subscription_faq.md

Relevant policy:

"Refunds are allowed within 7 days of purchase."

## Human Approval Point

Refunds above $500 require manager approval.

Purchase amount: $29

Approval required: No

## Final Output

Customer informed that refund is eligible and refund workflow initiated.

## Outcome

Ticket resolved automatically.

Estimated support time saved: 8 minutes.
